# FrostyFlow 模拟运行环境说明

## 概述

本项目提供了一个完整的模拟运行环境，用于测试UI/UX而无需连接到真实的区块链网络。所有功能都通过模拟服务(mockService)实现，可以完整展示前端界面和交互流程。

## 功能特性

1. **资产总览页面** - 显示用户持有的各种资产和收益情况
2. **质押铸造页面** - 模拟将原生代币质押为流动性质押代币的过程
3. **质押赎回页面** - 模拟赎回流动性质押代币获取原生代币和收益的过程
4. **响应式设计** - 支持桌面端和移动端访问
5. **Uniswap风格主题** - 现代化的深色主题界面，包含毛玻璃效果和渐变色彩
6. **移动端优化** - 为移动设备提供专门的导航抽屉和优化的布局

## 运行模拟环境

### 启动开发服务器

```bash
npm run dev
```

服务器将在 http://localhost:5174 启动（如果5173端口被占用，会自动使用下一个可用端口）

### 访问页面

1. **资产总览** - http://localhost:5174/
2. **质押铸造** - http://localhost:5174/stake
3. **质押赎回** - http://localhost:5174/redeem

## 模拟服务说明

所有模拟数据由 [mockService.js](src/services/mockService.js) 提供，包括：

- 模拟钱包连接
- 模拟资产数据
- 模拟质押和赎回操作
- 模拟收益计算
- 模拟交易历史

## 测试流程

1. **资产总览页面**
   - 自动加载模拟资产数据
   - 显示总资产价值和累计收益
   - 展示不同资产的详细信息

2. **质押铸造页面**
   - 选择要质押的资产(DOT/GLMR)
   - 输入质押数量
   - 实时查看收益预估
   - 点击"确认质押"按钮提交模拟交易

3. **质押赎回页面**
   - 选择要赎回的资产(vDOT/vGLMR)
   - 选择赎回类型(标准赎回/即时赎回)
   - 输入赎回数量
   - 查看赎回预览（包括手续费和到账金额）
   - 点击"确认赎回"按钮提交模拟交易

## 技术栈

- React 18
- Ant Design 5.x
- Vite
- React Router v6

## 响应式设计特性

- **桌面端**: 使用侧边栏导航，提供完整的功能访问
- **移动端**: 使用抽屉式导航菜单，优化触摸操作体验
- **自适应布局**: 根据屏幕尺寸自动调整组件大小和排列

## 注意事项

- 所有操作都是模拟的，不会产生真实的区块链交易
- 模拟服务提供延迟以模拟真实网络请求
- 可以通过修改 [mockService.js](src/services/mockService.js) 来调整模拟数据
- UI采用现代化的深色主题设计，具有良好的视觉效果和用户体验
